Проект python с использованием фреймворка DJANGO

Представляет таблицу строки которой можно добовлять и удалять для удобства каждая строка нуммеруется.
Все изменения которые произвел пользователь с таблицей сохраняются в БД так что после перезагрузки веб-сервера
пользователь видел свои изменения. Для взаимодействия с этим процесом в шаблоне предусмотренно две кнопки "добавить" и "удалить".
